  ## 改善したい点
  *ドローンの最大速度をvmaxとしたとき、加速度はa=log(vmax-v)みたいな感じになる。要は加速度は低速では大きくなるが、高速では小さくなるようにする。
  *同じドローンが二回連続ボールに触れないようにする。
  *スパイクの機構として、ボールに対して鉛直下向きの力をかけれるように、上から下に当たる。
  *スパイクするドローンは上でホバリングしてる
  *２チーム分のドローンを取りあえず二機用意し、お互いがそれぞれのコート内に入らないようにする。
  * 打撃する球の落下地点予測する機構をつける
  * 最大速度を制限する。
  * スパイクは相手チームのそれぞれのドローンが一番取りづらいところ、取り敢えずはy座標０で二つのドローンの最大二乗和になる箇所をねらうようにする。
  * できれば、ドローンは最大速度でスパイクする。
  * ドローンは相手の球は自コート内に入る可能性のある球のみ反応するようにする
  * 球がコート内外に接触したときは球を消す。
  * 点数をどっかに書く。直前に相手が触った球がコート内に接触(→相手に１点)、コート外に接触(→自分に１点)など
  

  ## スパイクアクション設計仕様 (Spike Action Design)

### 1. 設計思想：対戦相手に基づいた意思決定
スパイクは、レシーブやトスのような「味方との連携を重視する協調動作」とは根本的に異なり、対戦相手との駆け引きを重視します。

* **基本定義**: 相手ドローンの配置や隙間をリアルタイムに解析し、最適な打撃コースを決定する「対戦相手依存の戦略動作」と位置づけます。
* **視点の転換**: 味方の位置関係ではなく、相手ドローンの位置関係から打ち込むコースを決定します。

### 2. スパイク・シーケンス（基本アルゴリズム）
以下のステップで動作を構成します。

1.  **ポジショニング（陣取り）**
    * トスによって上がる球を打てる最適な位置を予測し、先んじて移動・待機します。
2.  **コース推測（コース決定）**
    * 相手ドローンの配置から、相手が「打たれて嫌な場所（デッドスペース）」を推測します。
3.  **打撃実行（移動とショット）**
    * 狙った場所へ飛ばすために必要な「速度」と「方向」を割り出し、打撃ポイントへ移動して接触します。



### 3. 物理・弾道計算の仕様
実装の初期段階をスムーズに進めるため、以下の前提条件を設けます。

* **打撃時のエネルギー**: 
    * 計算を簡略化するため、ドローン接触時のボールの初速度はゼロ（静止状態）と仮定します。
    * 弾道は、ドローンから加わる力と、設定された重力加速度から算出します。
* **絶対条件**: 
    * 放たれた球は必ずネットを超え、指定された相手コートの範囲内に届くように計算します。

### 4. センシング（データ取得）
開発のフェーズに合わせて、取得手法を段階的に移行します。

* **プロトタイプ（現在）**: 
    * ボールオブジェクトの `Transform.position` から直接座標を取得します。
* **本実装（今後）**: 
    * Raycast` 等のセンサーを用いた検知システムに移行します。

### 5.実装案
  *打ちたいコースを決める→弾道計算より球に当たる時の速度ベクトルを考える→その速度ベクトルを出せるように移動＆加速する。

  ## 打撃する球の落下地点予測
変数の定義
打撃地点:(x0,y0,z0)
落下地点:(x,0,z)
ドローンから球へ与える速度ベクトル:(vx0,vy0,vz0)

t秒での速度ベクトル:(vx0,vy0+g*t,vz0)

球の半径：r
余裕をもってネットを超す高さ（安全高度)：s
ネットの高さ:h
ネット上部での球の通過地点:(x2,y2,z2)
ネット上部の通過時刻:th

* **返球がコート内返せる条件**:
* 地面に達する時間tnを求める
*式： 0-y0=vy0*tn+0.5f*g*tn^2
* tn=(-vy±sqrt(vy^2-4*g*y0))/2*g
* t秒後に(vx0*tn,0,vz0*tn)に着地
* (x1<=x0+vx0*tn<=x2)&&(z1<=z0+vz0*tn<=z2)
* ⇔{(x1-x0)/tn<=vx0<=(x2-x0)/tn} && {(z1-z0)/tn<=vz0<=(z2-z0)/tn}//コート内に球が収まるドローンによって与えられた速度の範囲
* 

* **ネットを超える条件**:
* th=(x0-x2)/vx0
* ネット超える⇔h+<y2より
* ⇔h+s<y0+vy0*th+0.5f*g*th^2
* ⇔vy0<(-(h+s)+y0+0.5f*g*th^2)/th
* ⇔vy0<(y0-h-s)/th+0.5f*g*th

* よって、ドローンから球へ与える速度ベクトル(vx0,vy0,vz0)が以下の条件を満たすときコート内にボールが収まる。
* (x1-x0)/tn<=vx0<=(x2-x0)/tn　&&　vy0<(y0-h-s)/th+0.5f*g*th　&& (z1-z0)/tn<=vz0<=(z2-z0)/tn
